  2. 连接和发送

    • open() 传入三个参数：请求方式 (get | post)、请求地址、是否异步请求

    • GET 请求方式通过url将参数传递到服务器，POST 则是通过将数据作为 send 的参数传递到服务器

      • ps：POST 请求时，在发送数据之前，要设置表单提交的内容类型
      • 提交到服务器的数据先用 encodeURIComponent() 进行编码，防止特殊字符
        decodeURIComponent() 解码

  3. 接收

    • 接收到响应后，响应的数据会自动填充 XHR 对象

      • responseText 响应返回的主体内容，字符串类型
      • responseXML 响应主体为 "text/xml" or "application/xml"时，保存的数据为 xml 数据，为 XML 对应的 document 类型
      • status 响应的 HTTP 状态码
      • statusText HTTP状态码的说明

    • XHR 对象的 readyState 属性表示请求/响应过程的当前活动阶段，

      • 0 未初始化时，还没有开始调用 open() 方法
      • 1 启动，调用了 open() 方法，未调用 send()方法
      • 3 接收，已经接收到部分响应数据
      • 4 完成，已经接收到全部响应数据
