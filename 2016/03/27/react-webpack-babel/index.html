react-js开发环境

  • 开始项目
    npm init

*安装webpack


npm install webpack –save-dev

  • 项目目录
    1. /app
      1. main.js
      2. component.js
    2. /build
      1. bundle.js
      2. index.html
    3. webpack.config.js
    4. package.json
  • 设置webpack.config.js
    var path=require('path');
module.exports={
entry:path.resolve(__dirname,'app/main.js'),
output:{path:path.resolve(__dirname,'build'),
filename:'bundle..js'}};
  • 在package.json里面预设这个命令去启动打包功能
    'build':'webpack'
  • 使用webpack-dev-server:时时监听
    npm install webpack-dev-server --save-dev
  • 在package.json里面配置webpack-dev-server
    'dev':'webpack-dev-server --devtool eval --progress --colors --hot --content-base build'
注释: 1、webpack-dev-server 创建服务器端口
    2、devtool eval 创建源地址，使代码可以快速定位
    3、progress 显示进度条
    4、colors 使命令行带颜色
    5、content-base build 指向设置的输出目录
    
    时时监听
  • 启动自动刷新

    entry:[‘webpack/hot/dev-server’,’webpack-dev-server/client?http://lacalhost:4040‘,
    path.resolve(__dirname.’app/main.js’)]

    这才是重点:配置react/babel

    • 安装react

      npm install react –save

npm install ract-dom –save

  • 安装babel-loader
    npm install babel-loader –save-dev

npm install babel-core -save-dev

  • 安装支持ES5
    npm install babel-preset-es2015 –save-dev
  • 安装支持JSX
    npm install babel-preset-react –save-dev
    • 安装支持最新ES版本
      npm install babel-preset-stage-0 –save-dev
    • 安装Polyfilla
      npm install babel-polyfill –save
    • 减少重复代码
      npm install babel-runtime –save

npm install babel-plugin-transform-runtime –save-dev

  • 配置babel

    在入口添加polyfill

    entry[
    ‘babel-polyfill’,’webpack/hot/dev-server’,
    ‘webpack-dev-server/client?http://localhost:1010‘,
    path,resolve(dirname,’app/main.js’)]

    ‘es2015’,’stage-0’,’react’]
    }
